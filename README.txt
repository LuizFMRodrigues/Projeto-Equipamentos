----------------Menu-------------------
Primeiro é preciso cadastrar um funcionário antes de cadastrar um pedido.

----------------Cadastrar Funcionários-------------------
Ao clicar em salvar, uma pasta (c:\Projeto Equipamentos\) será criada. Esta pasta guardará todos os dados
dos pedidos e funcionários.

----------------Cadastrar Pedidos-------------------
Aqui são informados os dados do cliente e do pedido. O campo "Nome/Código do Pedido" será o nome do arquivo
txt salvo na pasta c:\Projeto Equipamentos\Pedidos. Ao clicar em "Salvar", deve-se informar o código do
funcionário cadastrado que foi salvo na pasta c:\Projeto Equipamentos\Funcionários.

Para atualizar o pedido basta informar o mesmo nome/código do pedido. Ao mudar o status do pedido o sistema
também irá mudar o nome do arquivo para informar que o pedido já foi finalizado.

----------------Visualizar Pedidos-------------------
Assim que clicar em visualizar, a lista será carregada com todos os pedidos salvos. Para filtrar basta alterar
o valor da ComboBox e clicar em "Atualizar".
